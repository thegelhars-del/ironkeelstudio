Project 01 · Flagship

Virginia-Class Block IV

Attack Submarine · SSN

In Development
[ VIRGINIA-CLASS HERO PHOTO ]

The flagship project. Built from original CAD files commissioned specifically for Iron Keel, the Virginia-class Block IV is the reference design the rest of the fleet is measured against.

Three sizes are in active development — 3", 4", and 6" diameter hulls — all sharing the same master geometry and scaling up through the Keel Core hybrid system.

Project 02

Columbia-Class

Ballistic Missile Submarine · SSBN

In Development

The next generation of American ballistic missile submarines, built on the 4" Keel Core system using a licensed hull design.

The Columbia-class is a larger boat than the Virginia, which makes it an ideal match for the 4" hull size — the scale works out and the full-length model holds presence without becoming unwieldy.

Project 03 · Custom

Block VI Custom Variants

Commemorative Builds

In Development
[ CUSTOM VARIANT HERO PHOTO ]

Commemorative and custom builds — hull numbers, specific paint schemes, unique features like the Cribbage Payload Module. Built for sailors, wardrooms, retirements, commissionings, and families who want a model that belongs to a specific boat.

Block VI adds features that weren't on earlier Virginia blocks. The customs can take that further — hull number, distinguishing marks, anything that makes the model the right boat instead of a generic boat.
